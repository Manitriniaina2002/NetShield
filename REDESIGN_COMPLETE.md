## 💡 How to Use

### Start Using New Components
```jsx
import { Button } from './ui/button'
import { Input } from './ui/input'
import { Badge } from './ui/badge'
import { StatsCard } from './ui/stats-card'
import { Card, CardHeader, CardTitle, CardContent } from './ui/card'

// In your component
<Button variant="primary" size="md">Click me</Button>
<Input placeholder="Enter text" />
<Badge variant="success">Active</Badge>
<StatsCard title="Metric" value="100" color="primary" />
```

### Apply to Existing Components
The new styling system is backward compatible. Existing components automatically benefit from the new CSS classes:
```css
.card              /* Modern card styling */
.btn-primary      /* Modern button classes */
.input            /* Modern input styling */
.badge            /* Modern badge styling */
```

### Responsive Design
Use Tailwind's responsive utilities:
```jsx
<div className="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-4 gap-4">
  {/* Responsive grid */}
</div>
```
